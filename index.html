0 먼저 알아둘 단어

아래에서 자주 나오는 노션 용어 몇 개입니다. 외울 필요는 없고, 한 번 훑어두면 뒷부분이 쉽게 읽힙니다.

데이터베이스(DB) — 프로젝트를 한 줄씩 모아 둔 표예요. 한 줄(=한 항목)이 건물 하나라고 생각하면 됩니다.
항목 / 페이지 — 표의 한 줄을 열면 그 안에 사진·설명을 쓸 수 있는 페이지가 나와요. 이게 건물 하나의 상세 페이지입니다.
속성(Property) — 항목 위쪽에 있는 “칸”들이에요. 건물명, 준공, 위치처럼 정해진 정보를 넣는 자리입니다.
템플릿 — 새 항목을 만들 때 고르면, 빈칸과 안내가 미리 채워져 나오는 양식이에요. “작성 가이드” 템플릿을 고르면 편합니다.
개발 / 운영 — “개발”은 연습장(작업·검토용), “운영”은 실제 손님이 보는 곳이에요. 먼저 개발에서 만들고, 다 되면 운영으로 옮깁니다.

1 전체 흐름 한눈에 보기

전체 순서는 다음과 같습니다. 개발에서 작성 → 확인 → 운영으로 이동 → 배포 요청.

STEP 1 · 개발

개발에서 작성

“개발” 표에서 새 항목을 추가하고 작성 가이드 템플릿을 선택합니다. 칸(속성)과 본문을 채웁니다. 여기서는 자유롭게 수정해도 됩니다.

→
STEP 2 · 확인

빠진 칸 확인

건물명·썸네일·준공·위치·발주처·유형·좌표(x/y)가 모두 채워졌는지 확인합니다. 특히 좌표 두 개가 서로 바뀌지 않았는지 점검하세요.

→
STEP 3 · 운영

운영으로 이동

확인이 끝나면 항목을 “운영” 표로 옮깁니다. 옮길 줄에서 우클릭 → 이동(옮기기) → 운영 선택. 운영은 실제 공개 영역이니 옮길 때는 신중하게 확인하세요.

→
STEP 4 · 배포

배포 요청

마지막으로 가동재 매니저에게 배포(반영)를 요청합니다. 배포해야 홈페이지에 반영됩니다. (아래 2번 꼭 확인)

① 내가 내용을 입력하는 곳 — 노션 표 바로가기

아래 버튼을 누르면 작업할 노션 표가 바로 열려요. 보통 개발에서 만들고, 다 되면 운영으로 옮깁니다.

🔑
이 노션 표들은 초대 권한을 받은 메일(예: design@hdc-labs.com)로만 열 수 있어요.
버튼을 눌렀는데 안 열리거나 “권한 없음”이 뜨면, 아직 초대받지 못한 거예요. 프롭테크운영 UNIT 가동재 매니저에게 초대를 요청해 주세요.

2 ⚠️ 제일 중요 — 노션에 쓴다고 바로 안 보여요

🚨
노션에 입력해도 사이트에 즉시 반영되지 않습니다. 사이트는 매일 한 번 자동으로 배포(반영)됩니다.
그래서 바로 확인해야 하거나 급하게 반영해야 할 때는 가동재 매니저에게 배포(반영) 요청을 해주세요.
⛔
절대 운영에 먼저 작성하지 마세요. 작성은 항상 개발에서 합니다.
운영은 매일 자동 배포되기 때문에, 운영에 미완성 상태로 글을 남겨 두면 그 미완성 내용이 그대로 사이트에 노출되는 경우가 있습니다. 반드시 개발에서 끝까지 완성한 뒤, 다 된 것만 운영으로 옮기세요.
이 수칙을 어겨 발생한 노출 사고는 작성한 당사자에게 책임을 묻겠습니다. 모든 작성·수정·이동 로그와 기록은 전부 확인할 수 있습니다.

개발 → 운영으로 옮기는 방법

  1. 개발에서 내용을 끝까지 완성합니다.
  2. 옮길 줄(항목)에서 우클릭 → 이동(옮기기) → 운영을 선택합니다.
  3. 운영은 공개 영역이니 옮기기 전에 한 번 더 신중하게 확인하세요. 옮긴 뒤에는 다음 자동 배포 때 사이트에 반영됩니다.

급할 때 배포 요청은 이렇게

  1. 노션에서 내용 입력(또는 운영으로 이동)을 끝냅니다.
  2. 가동재 매니저에게 “어떤 건물을 개발/운영 중 어디에 반영해 달라”고 요청합니다.
  3. 배포가 끝나면 위 사이트(개발 또는 운영)에서 실제로 보이는지 확인합니다.
📌
예시 메시지: “가동재 매니저님, ‘HDC랩스 사옥’ 개발에 반영 부탁드립니다.” / “‘VOCO 호텔’ 운영 배포 요청드려요.”

4 좌표 찍기 — x좌표 / y좌표 넣기

건물이 지도에서 어디 있는지 알려주는 숫자예요. 아래 사이트에서 위치를 찍으면 숫자 두 개가 나오는데, 그걸 그대로 노션에 옮겨 적으면 됩니다.

① 좌표 사이트 열기

🗺️ 좌표 찾는 사이트 열기 ↗
  1. 위 버튼을 누르면 새 창에 지도 사이트가 열려요.
  2. 지도에서 건물 위치를 찾아 클릭하면 숫자(경도·위도)가 나옵니다.
  3. 경도 숫자는 노션의 x좌표 칸에, 위도 숫자는 y좌표 칸에 넣어요.
💡
두 숫자를 바꿔 넣지 않도록 조심! x좌표 = 경도(126~129 정도), y좌표 = 위도(33~38 정도)예요. 헷갈리면 아래 도우미에 넣어 확인해 보세요.
🛠️
이 지도 좌표 페이지는 가동재 매니저가 직접 만든 도구입니다. 사이트가 안 열리거나 좌표가 이상하게 나오는 등 문제가 있으면 가동재 매니저에게 문의해 주세요.

5 마감 디테일 연결하기

마감 디테일은 “이 건물에 어떤 마감재(타일, 우드, 대리석 등)를 썼는지”를 따로 모아 둔 표(마감디테일 관계형)와 연결하는 칸이에요. 자료를 한 번 만들어 두면 여러 프로젝트에서 재사용할 수 있어요.

마감디테일 표는 이렇게 생겼어요

칸 이름무엇을 넣나요
이름마감재/디테일 이름. 예: 로비 대리석, 객실 우드패널, 외장 커튼월 디테일.
파일과 미디어그 마감재의 사진, 도면, 스펙 파일을 첨부해요.

연결하는 순서

  1. 먼저 마감디테일 관계형 표에 마감재 항목을 만들고(이름 + 파일 첨부) 저장해요.
  2. 개발(또는 운영) 프로젝트 항목을 열고 마감 디테일 칸을 클릭해요.
  3. 방금 만든 마감재 이름을 검색해서 골라요. 여러 개 연결해도 됩니다.
♻️
같은 마감재가 다른 프로젝트에도 쓰였다면, 새로 만들지 말고 이미 있는 항목을 골라 연결하세요. 자료가 한곳에 모여 관리가 편해집니다.
⛔
운영에서 쓰고 있는 마감디테일 항목은 함부로 지우지 마세요!
마감디테일은 여러 프로젝트가 함께 연결해 쓰는 자료예요. 운영(실제 공개 중)인 건물이 연결해 둔 항목을 지우면, 그 건물 페이지에서 마감 자료가 사라집니다. 지워도 되는지 확실하지 않으면 먼저 가동재 매니저에게 확인해 주세요.

6 상세 내용(본문) 쓰는 법

건물명 칸에는 짧은 이름만 넣고, 긴 설명은 항목을 “열기”로 펼쳐서 그 안에 작성합니다.

상세 패널 여는 법

  1. 표에서 작성할 건물명 위에 마우스를 올립니다.
  2. 이름 옆(또는 칸 안)에 나타나는 “열기” 버튼을 클릭합니다.
  3. 오른쪽에 상세 글을 쓸 수 있는 패널(페이지)이 열립니다. 여기 본문 영역에 내용을 작성합니다.

본문에 쓰면 좋은 순서

  • 1. 프로젝트 개요 — 어떤 프로젝트인지, 규모(연면적/층수), 우리 회사가 한 일(설계/시공/마감 등)을 한두 문단으로.
  • 2. 주요 공간 / 마감 포인트 — 로비·객실·라운지 같은 공간별로 핵심 마감재와 콘셉트를 정리. 사진과 함께 넣으면 좋아요.
  • 3. 이미지 갤러리 — 외관/내부 사진을 추가하고, 사진마다 캡션(어디 사진인지)을 달아주세요.
  • 4. 비고 / 특이사항 — 특별한 점, 수상 이력, 협력사 등 추가 정보.
✅
본문까지 다 작성했으면 운영으로 옮기고, 가동재 매니저에게 배포를 요청하세요. 배포해야 사이트에 반영됩니다. (위 2번 참고)
